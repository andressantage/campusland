EJERCICIO:
Dada unos números enteros, se pide calcular e imprimir:
• Cuáles y cuántos números son pares
• Cuáles y cuántos números son impares
La lista se termina cuando el número ingresado es -1 (Bandera)

ENTRADA: numero
PROCESO: ciclo: proceso aca da cada numero - WHILE mientras el numero no sea -1
        condicional: saber si es par o impar
SALIDA: el numero es par o impar
        cantidadPares, cantidadImpares, Contador

Pseint:
Algoritmo numeros_pares_impares
    Escribir "Numero entero: "
    Leer numeroEntero
    cantidadPares=0
    cantidadImpares=0
    Mientras numeroEntero!=-1 Hacer
        Si numeroEntero%2==0 Entonces 
            Escribir numeroEntero, " es PAR"
            cantidadPares=cantidadPares+1
        SiNo
            Escribir numeroEntero," es IMPAR"
            cantidadImpares=cantidadImpares+1
        FinSi
        Escribir "Numero entero: "
        Leer numeroEntero
    FinMientras
    Escribir "Cantidad de pares es: ", cantidadPares
    Escribir "Cantidad de impares es: ", cantidadImpares
FinAlgoritmo

Python:
pares = 0
impares = 0
numero = int(input("Digite un número entero: "))
while numero != -1:
    if numero%2 == 0:
        print("El número ",numero," es par")
        pares += 1
    else:
        print("El número ",numero," es impar")
        impares += 1
        
    numero = int(input("Digite un número entero: "))

print(f"Los números pares son: {pares}\nLos números impares son: {impares}")
